

🧠 Memory Matching Game

A fun and interactive memory card game built with React.js, where players flip cards to find matching pairs. Test your memory and concentration by matching all the tiles in the shortest time possible!

🎮 Features
	•	🃏 Flip Cards – Click on cards to reveal hidden images
	•	🔄 Match Pairs – Match two identical cards to keep them flipped
	•	❌ Auto Flip Back – Non-matching cards flip back after a short delay
	•	🏆 Win Condition – Game ends when all pairs are matched
	•	🎨 Custom Themes – Cards feature fruits
	•	⚡ Responsive Design – Works across different screen sizes

🛠️ Built With
	•	React.js
	•	CSS
	•	JavaScript (ES6+)

🧩 How It Works
	•	The game initializes with a shuffled set of card pairs
	•	Players can flip two cards at a time
	•	If the cards match → they remain visible
	•	If they don’t match → they flip back after a delay
	•	The game continues until all pairs are matched
